Gulp

WARNING: all listed below valid for Gulp 3!!!

Install Gulp

You have to install Gulp globally(for your computer) and locally(for your current project).

npm install gulp-cli --global

npm install gulp --save-dev

This will install the newest versions of code for the moment. If you need some exact version:

npm install gulp-cli@1.0.1 --global

npm install gulp@3.9.2 --save-dev

@1.0.1 - is a version's number. The same rule will work for all Gulp plugins.

Gulp-watch

Gulp-watch - can watch specific files and when it detects changes in this files it can trigger a gulp.task() in response.

Install gulp-watch

npm install gulp-watch --save-dev

Then you need to import it in the gulpfile.js

var watch = require("gulp-watch");

Gulp-watch - can watch different files and trigger different tasks.

Creating watch.task

var watch = require("gulp-watch");

gulp.task("watch", function() {

   watch("./app/index.html", function() {
      gulp.start("html");
   });

   watch("./app/assets/styles/styles.css", function() {
      gulp.start("styles");
   });
});

To run the code type - gulp watch. It keeps working until you stop it ctrl + c

Some important terms

gulp.src(); - points to the location of files to work with.

gulp.pipe() - any transformation or action we do to theese files.

gulp.dest(); - points to the location of result file.

Install Gulp-postcss

npm install gulp-postcss --save-dev

Import postcss to the gulpfile.js

var postcss = require("gulp-postcss");

Postcss by itself dosn't do anything. It needs plugins or filters. Let us install and import autoprefixer.

npm install autoprefixer --save-dev

var autoprefixer = require("autoprefixer");

In the same manner we can use postcss-simple-vars. It allows to use variables when writing a css code. And postcss-nested - allows nest css selectors.

npm install postcss-simple-vars --save-dev

npm install postcss-nested --save-dev
