Node&NPM

Install Node.js

Node.js makes JavaScript a fully functional programming language, so it can run not in a browser window only but also on local computers and servers operating with files and data bases.

Follow the Node Homepage (https://nodejs.org/) and download it LTS version. Run the instalation file.

To check if the instalation was successful type in the command line node -v.

"Out-of-the-box" Node.js has a bunch of modules for different purposes.

Some examples

Lets create a JavaScript file test.js

var fs = require('fs');

fs.writeFile(__dirname + "/index.html", "<h1>HTML is Great!</h1>", function(error) {
   if (error) {
      return console.log(error);
   } else {
      return console.log('Congrats!');
   }
});

After you run the node test.js in the command line this code will create new html file in the working directory with a level one title in it.

The code below will also download an image from given url and save it in local file.

var fs = require('fs'),
https = require('https');

fs.writeFile(__dirname + "/index.html", "<h1>HTML is Great!</h1>", function(error) {
   if (error) {
      return console.log(error);
   } else {
      return console.log('Congrats!');
   }
});

var myPhotoLocation = 'https://raw.githubusercontent.com/LearnWebCode/welcome-to-git/master/images/dog.jpg';

https.get(myPhotoLocation, function(response) {
   response.pipe(fs.createWriteStream(__dirname + "/mydog.jpg"));
});
